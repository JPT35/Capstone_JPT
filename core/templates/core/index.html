This CRM project is written in python using the django framework. The CSS styling is written using the tailwind framework. The charts for client reports are generated using chart.js and javascript. Each lead added can be edited and converted to a client. Once a lead is converted to a client, the client fields for number of month order must be updated. From the dashboard, users can export a .csv file of all leads and clients in the database. The purpose of this tool is to creat a simple interface for small business owners to keep track of their monthy orders and thier leads and clients.
